# Liezyl B. Baguios — Portfolio

A one-page professional portfolio with animated section transitions. Navigate via the header; each section appears with a smooth transition instead of scrolling.

## Contents

- **Home** — Intro and quick links  
- **About** — Objectives and qualifications  
- **Experience** — Work history (Teleperformance, Shore Solutions, Philippine Prudential Guarantee)  
- **Education** — BS Information System, ABE College Bacolod  
- **Skills** — MS Office, communication, sales, teamwork  
- **Interests** — Volunteer work, writing, sports, art & music  
- **Contact** — Address, email, phone  

## Run locally

Open `index.html` in a browser, or use a simple static server:

```bash
npx serve .
```

## Tech

- HTML, CSS, JavaScript  
- No build step; works in modern browsers
